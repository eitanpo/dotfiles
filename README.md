Init
====
```bash
chsh -s /bin/bash
cd ~ && mkdir Projects && cd Projects
git clone https://github.com/eitanpo/dotfiles.git && cd dotfiles && source bootstrap.sh
```
> You might be asked to install Xcode Command Line Tools

Setup
=====
```bash
source brew.sh
source ~/.macos
```

Update
======
```bash
source ~/Projects/dotfiles/bootstrap.sh
```
