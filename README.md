Init
====

$ cd ~ && mkdir Projects && cd Projects

$ git clone https://github.com/eitanpo/dotfiles.git

$ cd dotfiles && source bootstrap.sh


Setup
=====
```bash
cd ~/.dotfiles
source link.setup
./.macos
./brew.sh
```

Update
======
$ cd ~/Projects

$ git clone https://github.com/eitanpo/dotfiles.git

$ cd dotfiles && source bootstrap.sh
